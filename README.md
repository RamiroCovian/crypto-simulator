## Requisitos de instalacion

* **Crear entorno virtual**
  ```
  Windows
  python -m venv env
  --------------------
  Mac/Linux
  python3 -m venv env
  ``````
* **Activar entorno virtual**
  ```
  Windows
  env\Scripts\activate
  --------------------
  Mac/Linux
  source env/bin/activate
  ``````

* **Instalar Requirements**
  ```
  pip install -r requirements.txt
  ```
